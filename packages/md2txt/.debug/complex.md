# 技术文档示例

## 简介

这是一个包含**多种元素**的技术文档，用于测试 Markdown 转纯文本的效果。

## 功能特性

### 1. 文本格式

支持以下格式：

- **粗体文本**
- *斜体文本*
- ~~删除线文本~~
- `行内代码`

### 2. 链接和图片

访问 [GitHub](https://github.com) 获取更多信息。

![示例图片](https://example.com/image.png)

### 3. 代码块

JavaScript 示例：

```javascript
function hello(name) {
  console.log(`Hello, ${name}!`);
}

hello("World");
```

Python 示例：

```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

### 4. 列表

有序列表：

1. 第一步：准备环境
2. 第二步：安装依赖
3. 第三步：运行程序

无序列表：

- Node.js >= 18
- pnpm >= 8
- TypeScript >= 5

| 特性 | 支持 | 说明 |
|------|------|------|
| Markdown | ✅ | 完全支持 |

## 总结

这个文档包含了常见的 Markdown 元素，转换为纯文本后应该：

1. ✅ 移除所有格式标记
2. ✅ 保留文本内容
3. ✅ 保持可读性
4. ✅ 清理多余空行
